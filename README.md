# Wanandroid_Learning

## 介绍
本文是基于 [玩Android 开放API](https://www.wanandroid.com/blog/show/2) 进行开发，采用Androidx + MVVM + DataBinding + Retrofit 的开发模式。主要功能正是展示各类Android技术文章，目前文章根据知识体系、完整项目、微信公众号进行分类。
